# Streamlit Toolkit – Moringa AI Capstone Project

## Overview
This project is part of the **Moringa AI Capstone**: *Prompt-Powered Kickstart: Building a Beginner’s Toolkit*.  
It demonstrates how to build a **simple interactive Streamlit app** using AI-generated guidance. The app allows users to input text and displays a greeting along with a sample line chart.

---

## Features
- Interactive user input
- Dynamic greeting message
- Simple data visualization (line chart)
- Easy-to-run Python Streamlit application

---

## System Requirements
- **Operating System:** Windows / Mac / Linux  
- **Python Version:** 3.9+  
- **Dependencies:**  
  - streamlit  
  - pandas  
  - matplotlib  

---

bash

bash
Copy code

bash
Copy code

Common Issues & Fixes
Issue	Cause	Solution

App not updating after code changes	Cache or browser issue	Clear cache with streamlit cache clear and refresh browser
